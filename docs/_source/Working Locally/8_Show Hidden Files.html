Show Hidden Files

Many files are used by our local installation that are "hidden files".

Hidden files are folders or files whose names start with a period. By default, these are not viewable in File Explorer/Finder.

The are numerous situations where you may need to view your hidden files, see the OS-specific instructions below on how to show hidden files.

  • Windows Users:
    • Turn on View Hidden Files and Folders in your File Explorer menu:
      • Microsoft Support Article
    • On the same settings page, also deselect "Hide extensions for known file types and click OK."
  • Mac Users:
    • Use this keyboard shortcut from inside finder: Cmd+Shift+.
    • Use it again to hide hidden files.
